## docker-mariadb
---

MariaDB docker image with RocksDB plugin.

## Build
---
```
cd < version >
```
```
docker build -t <user>/<tag>:<version> .
```

## Push
---
```
docker login
```
```
docker push <user>/<tag>:<version>
```
